## Getting Started

Step by step to run this app in your local

### Install packages

At your directory root, run:

```
cd backend
```

```
npm install
```

### Run Backend server

```
node index.js
```
<br>
At your directory root, run:

```
cd frontend
```

```
npm install
```

### Run Frontend server

```
npm start
```
